Welcome to the Thaumatorium!

This is my personal website, where I write my personal thoughts and post my little projects.

Note that this site is pure uncompressed HTML/CSS/JS, with some PNG and WEBP images to fluff shit up. I like to think that this is how the web is supposed to be, other than that these are static pages instead of built up by some (C#/JS/PHP/Java) framework.

If you want an example on how not to do HTML, take a look at Twitter with their <div><div><div>divs</div></div></div> nonsense. Silly Twitter! That's not how you should use HTML!

This site's author is based in the Netherlands. I like to take an unnormal take on things: Look at what the masses are doing, learn from them by NOT doing that and then find out where the good shit is at.

That doesn't necessarily mean that popular = bad. It means that most programmers don't *really* care about quality or speed - if not them, the company they work for.

Disclaimer: I'm not perfect, so this file will likely contain small mistakes. If you find any, please open a Github Issue, whether it's just a spelling mistake or something wrong with the HTML.
