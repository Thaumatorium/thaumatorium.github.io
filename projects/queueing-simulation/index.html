Queueing System Simulation

Poisson arrivals, configurable server pool c, finite total system capacity K, and switchable service-time distributions.

Controls

Service variability (CV) 0.8
Simulation speed 10×

Model Notes

K is total system capacity, including the 2 customers in service. If the system already has K customers, a new arrival is dropped.

The reported averages are based only on the measurement window after warm-up. Wait/system times are conditional on served customers, not dropped arrivals.

Replications

Batch mode re-runs the same scenario with deterministic seeds seed, seed+1, ... and reports 95% confidence intervals.
