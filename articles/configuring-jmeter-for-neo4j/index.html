Steps to Configure JMeter for Neo4j

  1. Create a Thread Group
  2. Create a JDBC Connection Configuration via Add > Config Element
  3. Create a JDBC Request via Add > Sampler
  4. Create a View Results Tree (this shows the response data in a tab) via Add > Listener
