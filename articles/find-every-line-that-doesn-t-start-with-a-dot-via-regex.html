Find every line that doesn’t start with a . via Regex

Published: 2018-05-17

Author: NostraDavid

^((?!^\.).)*$

I was trying to filter bootstrap.css from all non-classes, so I found this built this. It searches for all lines that don’t start with a dot.

Great to use in combination with Notepad++ replace function (Ctrl + H)

[:].*$

To filter everything from a : till the end of the line.
