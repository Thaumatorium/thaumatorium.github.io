Regex explanation

( ) = grouping; use $1-$9 in the replace window to refer to this. $0 is the whole regex
[ – ] = a range of choices; usually choices are [a-z], [A-Z], [0-9]
[abc] = a list of choices; same as (a|b|c)
^ = not (this only counts if ^ exists between [ ] at the start)
\x00 = 0
\x7F = 127
(?=…)	= Positive lookahead
(?!…) =	Negative lookahead
(?:…) =	Non-capturing group; doesn't get saves to $1-$9

0-127 is the available range of ASCII characters, so since we want all NON-ASCII characters, we’re using the ^ operator, thus we get all unicode (by excluding ASCII)
